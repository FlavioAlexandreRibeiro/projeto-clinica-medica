Descrição Completa do Projeto — Site de Clínica Médica

O projeto desenvolvido consiste na criação de um site institucional completo para uma clínica médica, utilizando apenas HTML, com foco na aplicação prática dos principais conceitos estudados no módulo. O objetivo principal foi construir uma estrutura web organizada, funcional e semântica, abordando recursos fundamentais do desenvolvimento front-end.

Durante o desenvolvimento do projeto, foram aplicados diversos conteúdos essenciais, como estruturação de páginas, formatação de textos, criação de formulários, inserção de mídias, utilização de tabelas e organização semântica do conteúdo.

O site foi estruturado seguindo um padrão visual previamente definido no arquivo template.html, mantendo consistência em todas as páginas através da divisão em quatro áreas principais:

Menu de navegação;
Cabeçalho (Header);
Área de conteúdo (Content);
Rodapé (Footer).
Estrutura Geral do Site

Todas as páginas do projeto compartilham a mesma base estrutural, proporcionando padronização visual e melhor experiência de navegação ao usuário. O menu principal permite acesso às quatro páginas obrigatórias do sistema:

Página Principal;
Sobre a Clínica;
Horário de Atendimento;
Contato.

O rodapé foi mantido em todas as páginas contendo informações de contato e identificação da clínica.

Página Principal

Na página inicial foi inserida uma imagem no cabeçalho para representar visualmente a clínica médica e tornar o site mais atrativo.

Na área de conteúdo foi adicionada uma breve apresentação institucional da clínica, descrevendo seus objetivos, qualidade no atendimento e compromisso com os pacientes.

Nesta etapa foram utilizados:

Estruturação de texto com títulos e parágrafos;
Inserção de imagens;
Organização semântica da página;
Links de navegação no menu.
Página “Sobre a Clínica”

A página “Sobre a Clínica” foi criada com uma imagem diferente no cabeçalho, reforçando a identidade visual da seção.

No conteúdo principal foi desenvolvido um texto descritivo detalhando:

História da clínica;
Missão e valores;
Especialidades médicas;
Diferenciais de atendimento;
Estrutura oferecida aos pacientes.

Nesta página foram aplicados recursos de:

Formatação textual;
Hierarquia de títulos;
Ênfase em palavras utilizando tags semânticas;
Separação e organização de conteúdo.
Página “Horário de Atendimento”

A página de horários apresenta uma nova imagem no cabeçalho e possui uma descrição breve dos serviços oferecidos pela clínica.

O principal recurso desenvolvido nesta seção foi a criação de uma tabela completa contendo:

Especialidades médicas;
Horários de atendimento durante a semana;
Funcionamento aos sábados;
Atendimento em feriados.

A tabela foi estruturada utilizando corretamente:

<table>;
<tr>;
<th>;
<td>.

Os serviços exibidos foram:

Clínica Geral;
Psicologia;
Pediatria;
Oftalmologia.

Essa etapa permitiu praticar:

Organização tabular de dados;
Estruturação visual de informações;
Legibilidade e alinhamento de conteúdo.
Página “Contato”

A página de contato recebeu uma imagem exclusiva no cabeçalho e concentrou as principais informações de comunicação da clínica.

Foram adicionados:

Telefone fixo;
Celular;
WhatsApp;
Endereço completo da clínica.

Também foi incorporado um iframe do Google Maps exibindo a localização da clínica em tempo real, permitindo ao usuário visualizar o endereço diretamente no site.

Além disso, foi desenvolvido um formulário completo de contato contendo:

Campo de nome;
Campo de e-mail;
Campo de assunto;
Área de mensagem;
Botão de envio;
Botão de limpeza do formulário.

Nesta parte do projeto foram aplicados conceitos importantes relacionados a:

Formulários HTML;
Inputs de diferentes tipos;
Textarea;
Botões;
Integração de mapas externos via iframe.
Recursos e Conceitos Aplicados no Projeto

Durante a construção do site foram utilizados diversos recursos fundamentais do HTML, incluindo:

Estrutura básica de páginas HTML;
Tags semânticas;
Navegação entre páginas;
Inserção de imagens;
Criação de listas;
Formatação de textos;
Tabelas;
Formulários;
Incorporação de mídias externas;
Organização visual de conteúdo;
Padronização estrutural das páginas.
Resultado Final

Ao final do desenvolvimento, foi criado um site institucional funcional e organizado para uma clínica médica, demonstrando a aplicação prática de todos os conteúdos estudados no módulo de HTML.

O projeto proporcionou experiência na criação de múltiplas páginas interligadas, estruturação correta de conteúdo web e utilização de componentes essenciais presentes em sites profissionais.
